02 / Problem

행사장을 나서는 순간
당원의 연결이 사라진다

더불어민주당의 행사·집회·전당대회는 강력한 결집 경험을 만들지만, 당원이 귀가하는 순간 그 열기가 식어버립니다. 일상 속 감성 소통 채널이 없습니다.

현재 문제

① 행사 종료 = 연결 단절

전당대회, 집회, 행사의 뜨거운 열기는 귀가 후 급격히 식습니다. 일상에서 당과 연결되는 감성 채널이 없습니다.

현재 문제

② 일방적 문자·공지

현재 당원 접점은 SMS, 카카오 알림, 공지문 등 일방향에 그칩니다. 당원의 감정·상황에 반응하는 쌍방향 채널이 없습니다.

현재 문제

③ 핵심 당원 육성 수단 부재

당원이 지속적으로 참여하고 성장할 수 있는 콘텐츠·커뮤니티 플랫폼이 없습니다. 당원 이탈률이 높아집니다.

💡 기회의 크기

더불어민주당 등록 당원 약 150만 명.
이 중 핵심 활동 당원 약 20만 명.
라디오카페 연동 시 일 평균 당원 참여 시간 +24분 추가 가능.

▸ 라디오 기반 정치 소통 앱 평균 DAU 유지율 71%
▸ 오디오 콘텐츠 소비 시 정책 이해도 +38% 향상
▸ 당원 활동 포인트 적립 시 재참여율 89% (평균 대비 3배)

// 1. 감정 점수 계산 (현재 청취 트랙 t 기준) E(t) = w₁·BPM_norm(t) + w₂·Valence(t) + w₃·Energy(t) - w₄·Acousticness(t) // 가중치: w₁=0.25, w₂=0.35, w₃=0.30, w₄=0.10 / 모든 변수 0~1 정규화 // 2. 정치 이벤트 컨텍스트 보정 E_context(t) = E(t) × (1 + γ·ElectionProximity) × (1 + δ·PolicyHotness) // γ: 선거 D-30 이내 +0.25, 일반 시기 0 / δ: 핫이슈 정책 +0.15, 일반 정책 0 // 3. 당원 개인 활동 히스토리 반영 E_member(t) = E_context(t) × (1 - λ) + λ × MemberHistory_avg // λ = 0.35 (히스토리 반영률) / 신규 당원은 λ=0 // 4. 당원 활동 추천 최종 점수 Score(activity_j) = cos_sim(E_member(t), ActivityVector_j) × EngagementBoost_j × GradeMatch_j // ActivityVector: [참여난이도, 정치연관성, 사회관계성, 정보깊이, 행동유발력] 5차원 벡터 // GradeMatch: 당원 등급 매칭 계수 (일반당원=0.8, 핵심당원=1.0, 당직자=1.2) // 상위 3개 활동 → 당원 앱 맞춤 활동 추천 푸시 알림으로 전달
